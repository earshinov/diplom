Использованные в программе возможности, предоставляемые новым
стандартом языка программирования C++, известным как C++0x:

1. вывод типов (ключевое слово auto);
2. новая разновидность инструкции for для прохода по элементам коллекций;
3. двоичная запись целочисленных констант (префикс 0b);
4. возможность заполнения стандартных коллекций через списки
   инициализации (initializer lists);
5. анонимные функции (lambda);
6. rvalue references и связанные с ними новые функции стандартной
   библиотеки std::move и std::forward, с помощью которых реализуется
   семантика перемещения, позволяющая избежать накладных расходов на
   копирование содержимого коллекций и в то же время сохранить слабую
   связность классов (элементов программной системы на уровне кода),
   что облегчает их разработку и сопровождение.

Возможности 1-5 используются прежде всего для сокращения и улучшения
читабельности исходного кода.  Возможность 6 (rvalue reference)
позволяет использовать в коде семантику перемещения, которая по
сравнению с используемым ранее подходом на основе семантики копирования
позволяет сократить расходы на копирование в памяти содержимого
коллекций и таким образом улучшить скорость работы программы.  По
результатам простейшего тестирования производительности скорость работы
программы в режиме перебора конечных АВВП улучшилась на четверть.

Перечисленные возможности нового стандарта языка уже поддерживаются
современными версиями наиболее распространённых компиляторов для языка
C++, а именно Microsoft Visual C++ и GCC.
